## Description
This is a **web-based version** of the College Enquiry Chatbot using **HTML, CSS, and JavaScript**.  
It answers queries about courses, branches, fees, timings, and placements.

## Features
- Web-based GUI chatbot
- Rule-based answers for courses and branches
- Works in any browser
- Modern chat bubbles

## How to Run
1. Open `index.html` in any browser
2. Type queries like `course`, `CSE`, `fee`, or `timing`  
3. Chatbot responds instantly
